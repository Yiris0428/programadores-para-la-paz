¿Qué es el login en una aplicación?
El login es el proceso mediante el cual un usuario se identifica en una aplicación con sus
credenciales para acceder a sus funciones.

¿Por qué algunas rutas deben estar protegidas?
Algunas rutas deben estar protegidas porque pueden contener información privada,
registros internos o acciones que no deberían estar disponibles para cualquier persona.

¿Qué tipo de información no debería exponerse públicamente?
No se deben exponer públicamente contraseñas, documentos de identidad, teléfonos,
direcciones, tokens, claves de API ni otros datos sensibles de la comunidad.
